一、购买digitialocean的vps
	(1)进入https://cloud.digitalocean.com注册账号
	(2)创建droplets(和与服务器的ecs大体一样，只不过带宽很高)，需要支付，这里选择用paypal支付平台,paypal(具体的流程百度),然后进行支付(要求不是很高的话5$美元的那个配置稳稳够了)。
	(3)然后是选择操作系统(推荐centos)，还有服务器地区(看你具体主要翻哪个个地区的就选最近的服务器地区，相差不是很大)。注意:如果想以后支持ipv6线路的话下面有个Select additional options勾选Private networking和IPv6
	   这两个选项(以便以后想支持的话比较麻烦一点)，再下载个ssh工具对你买的服务器进行管理（管理比较简单）。
	(4)这里第一次登陆密码是一串随机数，会发送到你注册的邮箱，注意查看，第一次登陆时会提示修改root密码。至此，DigitalOcean服务器搭建完毕，接下来我们要在这个服务器上安装Shadowsocks。

二、进入droplet控制台搭建Shadowsocks
	(1)ShadowSocks服务器端配置
	   ①:ShadowSocks是一款高性能的全局socks v5代理软件，具有性能稳定、使用方便、多平台使用等特点，支持Windows,Linux,Android,iOS等(更具体的去百度)。
     	   ②:接下来详细介绍一下搭建步骤，首先是安装：
		Debian / Ubuntu下的命令:
		apt-get update
		apt-get install Python-pip	
		pip install shadowsocks

		CentOS下的命令:
		yum update
		yum install python-setuptools && easy_install pip
		pip install shadowsocks

		然后我们在/etc/目录下创建一个shadowsocks.json文件（vim /etc/shadowsocks.json），用来记录服务器端的配置信息，
		
		单用户访问格式如下：
		{
   		  "server":"droplet_ipv4"/"server":"droplet_ipv6"/"server":"::", 
    		  "server_port":8388,
   		  "local_address":"127.0.0.1",
   		  "local_port":1080,
    		  "password":"your_password",
   		  "timeout":300,
   		  "method":"aes-256-cfb",
    		  "fast_open": false
		}

		多用户访问格式如下
		{
    		  "server":"droplet_ipv4"/"server":"droplet_ipv6"/"server":"::",
    		  "local_address":"127.0.0.1",
    		  "local_port":1080,
    		  "port_password":{
        		"user1_port":"user1_password",
        		"user2_port":"user2_password"
    		},
    		  "timeout":300,
    		  "method":"aes-256-cfb",
    		  "fast_open": false
		}


		解释下几个字段：

		server填写你购买的服务器的ip地址（v4或v6），在网站管理界面，或者使用ifconfig命令都可以得到，这里也可以写成”server”:”::”，这样可以同时监听v4和v6,具体看你需求选择。

		server_port：监听的端口，保证端口不被占用即可。

		local_address：本地的地址。

		local_port：本地执行时的端口，主要用于客户端情况。

		method:加密方式，这里使用的是aes-256-cfb，也有其他的选项，对这块不了解就不多介绍了。

		设置完之后，我们可以用以下指令，在后台打开或关闭shadowsocks服务：

		ssserver -c /etc/shadowsocks.json -d start
		ssserver -c /etc/shadowsocks.json -d stop

		最后设置让shadowsocks.json开机自启动:
		vim /etc/rc.local
		在末尾加上ssserver -c /etc/shadowsocks.json -d start




		注意:如果以前已经打开过shasowsocks服务(即ssserver -c /etc/shadowsocks.json -d start命令执行过),当修改改了/etc/shadowsocks.json或变更了文件名后执行ssserver -c /etc/shadowsocks.json(新改的的) -d start
		会出现错误( ERROR already started at pid 1532)这是必须停止以前的shadowsocks服务(执行ssserver -c /etc/shadowsocks.json(旧的的) -d stop)，再执行ssserver -c /etc/shadowsocks.json(新改的的) -d start

	(2)ShadowSocks在windows的客户端端配置
	   ①这里先上shadowsocks客户端下载链接：https://www.shadowshocks.com/client.html 或 https://www.shadowsocks.com.hk/client.html(这两个链接是不一样的，不是线路不一样，
	     仔细看看，因为开源所以我也不知道真正是哪个网站，哪个下载的能用就用哪个吧)
           ②把得到最终让他形成shadowsocks.exe（具体看你下载的是源码还是压缩包，我的是压缩包，解压后直接是exe格式）
           ③点击运行shadowsocks.exe程序，在对应位置填写好服务器IP、端口、密码、加密方式等。代理端口和配置文件里面的local_port一样，然后确定，启用系统代理，这样全局代理就已
	     经生效了。然后是浏览器上的代理，这里以chrome的SwitchySharp为例(具体功能是一个代理插件)：我们新建一个情景模式，选择手动配置，只需要在SOCKS代理里面填写127.0.0.1,
	     端口填写1080（即local_port），然后选择socks v5就可以了。其他浏览器的配置也大同小异，网上对于这类软件的文章太多了，这里就不一一介绍了。
